# Примитивы

Помимо [модулей](/docs/modules), которые можно использовать с LCEL, GigaChain также включает различные примитивы, которые помогают передавать и обрабатывать данные, привязвать аргументы, вызывать пользовательскую логику и многое другое.

В этом разделе представлены некоторые примеры того, как примитивы могут помочь при разработке приложений с GigaChain.
